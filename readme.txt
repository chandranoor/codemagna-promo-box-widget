Provide featured boxes for selected contents. Useful if you want to highlight your content or links

== Description ==

Regenerate Thumbnails allows you to Provide featured boxes for selected contents with thumbnail image

This is useful for situations such as:

* You want to display link or url that your visitor should noticed
* You want to highlight or featuring your content

== Installation ==

1. Go to your admin area and select Plugins â Add New from the menu.
2. Search for "Codemagna Promo Box Widget".
3. Click install.
4. Click activate.
5. Navigate to Appearance â Widgets then drag CODEMAGNA - Promo Box to widget area

== Frequently Asked Questions ==

= Is this plugin [GDPR](https://en.wikipedia.org/wiki/General_Data_Protection_Regulation) compliant? =

This plugin does not log nor transmit any user data. Infact it doesn't even do anything on the user-facing part of your website, only in the admin area. This means it should be compliant but I'm not a lawyer.
